
# **AI-Powered Test Automation with Playwright**
Цей курс орієнтований на спеціалістів, які хочуть оволодіти сучасною автоматизацією тестування вебзастосунків на JavaScript/TypeScript, використовуючи потужний фреймворк **Playwright** та інструменти штучного інтелекту для прискорення та підвищення стабільності тестів.

Для навчання потрібно знати теорію тестування та основи JavaScript/TypeScript, HTML/CSS, Git, SQL.
Отримати ці навички можеш на курсах «Основи автоматизації тестування» та «Основи баз даних».


## **ЧОМУ ВАРТО ЗНАТИ АВТОМАТИЗАЦІЮ ТЕСТУВАННЯ:**
Автоматизація тестування є важливою складовою для швидкої та ефективної розробки програмного забезпечення.
Вона дозволяє випускати якісні продукти забезпечуючи тестування на всіх етапах розробки.
Автоматизовані тести можна повторно запускати, задавати різні набори даних, використовувати на різних платформах, виконувати автономно.
Всі ці чинники суттєво скорочують час перевірки програмного забезпечення.


## **ЧОМУ PLAYWRIGHT — ЦЕ СЬОГОДНІШНІЙ СТАНДАРТ АВТОМАТИЗАЦІЇ?**
Playwright — сучасний інструмент від Microsoft, створений як відповідь на обмеження Selenium.
Він підтримує автоматичні очікування, тестовий раннер, роботу з API, потужні репорти та AI-функції.
Його використовують провідні компанії, оскільки він забезпечує:

- стабільність тестів без зайвих очікувань та "sleep";
- швидке виконання та паралельність із коробки;
- нативну підтримку Chromium, WebKit, Firefox та мобільних емуляцій;
- перехоплення мережі, mocking, роботу з файлами та cookies;
- інтеграцію зі штучним інтелектом для self-healing та генерації тестів.


## **ПІСЛЯ ЗАВЕРШЕННЯ КУРСУ ТИ ЗМОЖЕШ:**
- будувати повноцінні фреймворки Playwright з нуля;

- застосовувати AI для генерації тестів, локаторів та POM;
- будувати Page Object та Component Object моделі;

- робити network mocking, routing, request interception;
- готувати проєкти до CI/CD та формувати професійну звітність;
- писати data-driven, parameterized та cross-browser тести;
- використовувати Allure для кроків, вкладень, скріншотів та відео.
